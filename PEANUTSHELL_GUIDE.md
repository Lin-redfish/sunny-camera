# 花生壳配置指南

## 第1步：注册花生壳账号

### 1.1 访问官网
- 网址：https://hsk.oray.com/
- 点击右上角"注册"

### 1.2 填写注册信息
- 手机号：使用你的手机号
- 验证码：接收短信验证码
- 密码：设置安全密码
- 确认密码：再次输入

### 1.3 登录账号
- 注册成功后登录
- 进入控制台

---

## 第2步：下载花生壳客户端

### 2.1 下载客户端
访问：https://hsk.oray.com/download

选择适合你系统的版本：
- Windows：下载Windows客户端
- Mac：下载Mac客户端
- Linux：下载Linux客户端

### 2.2 安装客户端
- 双击下载的安装包
- 按照提示完成安装
- 安装完成后启动客户端

### 2.3 登录客户端
- 使用注册的账号登录
- 确保登录成功

---

## 第3步：配置端口映射

### 3.1 添加映射
在花生壳客户端中：

1. **点击"添加映射"**
   - 找到"内网穿透"或"端口映射"选项
   - 点击"添加"或"+"按钮

2. **填写映射信息**
   - **应用名称**：camera-rental（或其他你喜欢的名字）
   - **应用类型**：HTTP
   - **内网主机**：127.0.0.1 或 localhost
   - **内网端口**：3000
   - **外网端口**：80（自动分配）
   - **绑定域名**：选择免费域名（例如：abc123.hsk.oray.com）

3. **保存配置**
   - 点击"保存"或"确定"
   - 等待映射创建成功

### 3.2 查看映射信息
创建成功后，你会看到：
- **外网访问地址**：http://abc123.hsk.oray.com
- **状态**：显示"在线"或"正常"

---

## 第4步：修改config.js

### 4.1 打开config.js
文件路径：`d:\相机租赁\config.js`

### 4.2 修改API地址
将：
```javascript
module.exports = {
  API_BASE_URL: 'http://192.168.14.156:3000/api'
}
```

改为：
```javascript
module.exports = {
  API_BASE_URL: 'http://abc123.hsk.oray.com/api'
}
```

**注意**：
- 将 `abc123.hsk.oray.com` 替换为你的实际域名
- 保留 `/api` 后缀

### 4.3 保存文件
- 保存config.js
- 在微信开发者工具中重新编译

## 第6步：配置小程序服务器域名

### 6.1 登录微信公众平台
- 网址：https://mp.weixin.qq.com/
- 使用你的小程序账号登录

### 6.2 进入服务器域名设置
- 点击左侧"开发"
- 选择"开发管理"
- 点击"开发设置"
- 找到"服务器域名"

### 6.3 添加合法域名
点击"修改"或"添加"，添加：

**request合法域名**：
```
http://abc123.hsk.oray.com
```

**注意**：
- 开发阶段可以勾选"不校验合法域名"
- 正式上线时必须配置合法域名
- 域名必须是花生壳分配的域名

## 常见问题

### Q1: 花生壳客户端显示离线？
A: 检查：
- 网络连接是否正常
- 账号是否登录
- 防火墙是否阻止

### Q2: 外网无法访问？
A: 检查：
- 本地服务是否运行（node server.js）
- 端口映射是否正确（3000端口）
- 花生壳客户端是否在线

### Q3: 免费版流量不够用？
A: 免费版限制：
- 流量：1GB/月
- 适合小规模使用

如果不够：
- 升级到付费版
- 或换用Serverless方案

### Q4: 每次重启电脑域名会变吗？
A: 免费版：
- 域名固定，不会变
- 但需要重新启动花生壳客户端

### Q5: 小程序显示网络错误？
A: 检查：
- config.js中的域名是否正确
- 花生壳客户端是否运行
- 后端服务是否运行
- 微信开发者工具是否关闭域名校验

## 注意事项

### 1. 电脑必须24小时开机
- 花生壳需要你的电脑在线
- 如果电脑关机，外网无法访问
- 建议设置电脑不休眠

### 2. 流量限制
- 免费版：1GB/月
- 适合小规模使用
- 超出后需要升级

### 3. 安全性
- 花生壳提供基础安全
- 建议定期修改密码
- 注意保护你的数据库密码

### 4. 稳定性
- 免费版可能偶尔不稳定
- 适合测试或小规模使用
- 正式上线建议使用Serverless
