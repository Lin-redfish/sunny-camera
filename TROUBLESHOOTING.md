## 问题排查

### 问题1：小程序显示"网络错误"

#### 原因分析
1. **协议不匹配**
   - config.js使用HTTPS
   - 但后端服务是HTTP
   - 导致连接失败

2. **域名无法访问**
   - 花生壳域名可能未配置
   - 或花生壳客户端未运行

3. **后端服务未运行**
   - 服务可能已停止
   - 需要重新启动

---

### 解决方案

#### 方案A：临时使用HTTP（测试用）

**步骤**：
1. 修改config.js为HTTP
```javascript
module.exports = {
  API_BASE_URL: 'http://localhost:3000/api'
}
```

2. 在微信开发者工具中关闭域名校验
   - 点击右上角"详情"
   - 选择"本地设置"
   - 勾选"不校验合法域名"

3. 重新编译小程序

**优点**：
- 可以快速测试
- 无需配置HTTPS

**缺点**：
- 只能在开发工具中使用
- 无法在手机上预览

---

#### 方案B：配置花生壳HTTPS（推荐）

**步骤**：
1. **升级花生壳到付费版**
   - 登录花生壳账号
   - 升级到标准版（68元/年）
   - 支持HTTPS

2. **重新配置端口映射**
   - 删除旧映射
   - 添加HTTPS映射
   - 获取HTTPS域名

3. **修改config.js**
```javascript
module.exports = {
  API_BASE_URL: 'https://你的花生壳域名/api'
}
```

4. **配置小程序服务器域名**
   - 登录微信公众平台
   - 添加request合法域名
   - 输入HTTPS域名

**优点**：
- 支持HTTPS
- 可以在手机上使用
- 稳定可靠

**缺点**：
- 需要付费（68元/年）

---

#### 方案C：使用Cloudflare Tunnel（免费HTTPS）

**步骤**：
1. **注册Cloudflare**
   - 访问：https://dash.cloudflare.com/sign-up
   - 注册免费账号

2. **下载cloudflared**
   - 访问：https://github.com/cloudflare/cloudflared/releases
   - 下载Windows版本

3. **运行cloudflared**
```bash
# 解压下载的文件
cd cloudflared

# 运行命令
cloudflared tunnel --url http://localhost:3000
```

4. **获取HTTPS域名**
运行后会显示：
```
https://abc123.trycloudflare.com
```

5. **修改config.js**
```javascript
module.exports = {
  API_BASE_URL: 'https://abc123.trycloudflare.com/api'
}
```

6. **配置小程序服务器域名**
   - 登录微信公众平台
   - 添加request合法域名
   - 输入HTTPS域名

**优点**：
- 完全免费
- 支持HTTPS
- 速度快

**缺点**：
- 域名会变
- 需要每次重启重新配置

## 常见错误及解决

### 错误1：request:fail
**原因**：网络请求失败

**解决**：
1. 检查API地址是否正确
2. 检查后端服务是否运行
3. 检查域名校验设置

### 错误2：request:fail url not in domain list
**原因**：域名未在白名单中

**解决**：
1. 登录微信公众平台
2. 添加request合法域名
3. 或关闭域名校验（开发阶段）

### 错误3：request:fail timeout
**原因**：请求超时

**解决**：
1. 检查网络连接
2. 检查后端服务响应时间
3. 增加超时时间
